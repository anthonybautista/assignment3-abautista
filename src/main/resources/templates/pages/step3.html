Step 3: Importing Avalanche Network

Now that your wallet is set up, it's time to add the Avalanche Network. Click the network dropdown (default network is Ethereum), and then click "Add Network".

Select Avalanche Network

Luckily, MetaMask recently added Avalanche as a pre-installed network. It used to be that you had to enter all sorts of information such as the RPC and block explorer URL, but now it's as simple as selecting Avalanche!

That's it! Now you can interact with dApps on the Avalanche Network!
